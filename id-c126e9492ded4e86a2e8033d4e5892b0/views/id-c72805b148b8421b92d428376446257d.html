Objetivos y Requerimientos SGDEA de EAAB ()
Con base en el diagnóstico de producción documental es importante plantear como prioridad incluir dentro del SGDEA la implementación del manejo de la siguiente documentación. El SGDEA es una definición de requisitos funcionales para un Sistema de Gestión de Documentos Electrónicos de Archivo. No es un patrón de diseño de software.
